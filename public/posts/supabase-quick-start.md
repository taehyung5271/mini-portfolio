# Supabase로 빠르게 백엔드 구축하기
## Supabase란?
Firebase의 오픈소스 대안이다. PostgreSQL 기반이라 관계형 데이터를 다루기 좋다.

## 시작하기
1. supabase.com에서 프로젝트 생성
2. Table Editor에서 테이블 만들기
3. supabase-js 설치 후 클라이언트 연결

## 데이터 조회
select 하나로 REST API 없이 바로 데이터를 가져올 수 있다. 프론트엔드에서 직접 DB에 접근하는 느낌이다.

## 장점
- 별도 백엔드 서버 없이 빠르게 시작 가능
- 실시간 구독 기능 내장
- Row Level Security로 보안 처리 가능
- PostgreSQL이라 관계형 데이터에 강하다
